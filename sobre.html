Sobre o site

O projeto Street Pet foi desenvolvido para ong “Street Pet”, localizada na cidade de Taquaritinga com intuito de facilitar a adoção legal pela internet. Com uma interface fácil e simplificada o projeto tem como objetivo ajudar os animais abandonados de rua com a realização de cadastro do adotador e pets, podendo ter a ficha completa do animal. Este projeto foi criado através da linguagem Hypertext Markup language (HTML), projetado e implementado na linguagem de programação “Visual Studio Code 2017”. Com a utilização do sistema, a população poderá realizar a adoção e o cadastro de animais de forma rápida e eficaz, possibilitando total segurança aos animais e adotadores.
